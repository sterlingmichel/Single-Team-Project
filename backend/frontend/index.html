This project was created by Sterling Michel using a range of opensource tool like Flask, Postgresql, JQuery, JQuery-UI, Swagger, CSS, Javascrripts and many others.

The project demostrate three key features.

  1. First, it allows a person to create, view and edit contact and allow ability to search and sort the diplay data.
  2. Second, it gives you the ability to display the contact address in a google map.
  3. Third, it enable to export a contact and share it.
